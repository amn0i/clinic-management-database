# Clinic Management Database

## Overview
This project presents the design and implementation of a Clinic Management Database using MySQL. It was developed as part of a university database course.

## Features
- Entity-Relationship Diagram (ERD)
- Relational Schema
- Database Normalization
- MySQL Tables
- SQL Queries
- Indexes
- Roles and Privileges
- Triggers

## Technologies
- MySQL
- SQL
